Automate License and Export Compliance Reporting.

Workflow:

  1. Clone scripts

    infra/license-compliance and infra/encryption-compliance

  2. Create local repositories with binary and source packages;
  3. Create License Compliance report with rpm and dpkg-deb utils;
  4. Create Export Compliance report via Black Duck Export. For rescan exist projects use FORCE_ANALYSIS;
  5. Merge License and Export reports;
  6. For remove exist project in Black Duck use REMOVE_PROJECTS;
