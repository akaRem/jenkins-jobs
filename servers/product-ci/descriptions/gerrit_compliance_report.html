Automate Export Compliance Reporting.

Workflow:

  1. Clone scripts infra/encryption-compliance

  2. Clone projects with source code from git repositories;
  3. Checkout to required branch and upload project to Black Duck;
  4. Create Export Compliance report via Black Duck Export. For rescan exist projects use FORCE_ANALYSIS;
  5. For remove exist project in Black Duck use REMOVE_PROJECTS;
  6. Currently this job does not publish report on drive.google.com
