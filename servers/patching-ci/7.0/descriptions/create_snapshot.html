Creates snapshot of repositories produced by build systemDISTRO

  1. get specified snapshot (SNAPSHOT_TIMESTAMP) from nearest location (from LOCATIONS)
  2. if DISTRO=="centos-6"
    • build second-level artifacts by triggering 7.0-build.late.artifacts job
    • push artifacts downloaded from artifacts.txt file into proposed repo on snapshot created on step 1
    • rebuild proposed repo (update repo metadata)
  3. sync updated local snapshot to all of the LOCATIONS as new one
    4. Docs: MOS Patching in a Nutshell.
