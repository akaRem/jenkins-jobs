This job runs set of swarm tests against 6.1 repositories. As 6.1 GA was released it is being used for acceptance and regression testing of code merged into stable branches which gets built into updates/proposed/security package repositories. It has following parameters:
  • MAGNET_LINK - Magnet Link to the ISO
  • UPGRADE_TARBALL_MAGNET_LINK - Upgrade tarball magnet Link
  • UBUNTU_LATEST - snapshots/ubuntu-2015-07-10-124412 relative path to /mos/.. containing mirror with packages
    of ubuntu to test against
  • RPM_LATEST - snapshots/centos-6-2015-07-10-123110 relative path to /mos/.. containing mirror with packages
    of RPM to test against
  • ENABLE_PROPOSED|SECURITY|UPDATES - whether to enable particular suites of packages - mirror of proposed/updates/security
    packages. This assumes enabling mos6.1-\ dists for ubuntu and "/\" subdirectory with packages for CentOS
    of RPM to test against
  • UBUNTU_MIRROR_ID - ID of ubuntu repository to test against. should be ubuntu-\
Docs: FUEL QA Docs , Mirantis OpenStack Patching in a Nutshell.
